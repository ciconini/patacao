# Shared Module

This directory contains shared code used across multiple modules.

## Structure

- **components/**: Reusable UI components following Atomic Design
  - `atoms/`: Basic building blocks (Button, Input, etc.)
  - `molecules/`: Simple component combinations (FormField, SearchBox, etc.)
  - `organisms/`: Complex components (Header, Sidebar, DataTable, etc.)
- **hooks/**: Shared Angular hooks and utilities
- **utils/**: Utility functions, formatters, validators
- **services/**: Shared services (API client wrapper, etc.)
- **stores/**: Shared state management (AuthStore, UIStore)
- **types/**: Shared TypeScript types and interfaces
